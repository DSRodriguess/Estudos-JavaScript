
# Estudos JavaScript

Este repositório contém estudos, exemplos e projetos simples desenvolvidos para aprofundar os conhecimentos em JavaScript puro.

## 📁 Estrutura do Projeto

O projeto está organizado de forma simples, com arquivos e pastas como:

```
Estudos-JavaScript/
├── index.html
├── scripts/
│   ├── script1.js
│   ├── script2.js
│   └── ...
├── styles/
│   ├── style.css
│   └── ...
└── README.md
```

## 🚀 Tecnologias Utilizadas

- JavaScript (ES6+)
- HTML5
- CSS3

## 🛠️ Como Executar

Basta abrir o arquivo `index.html` em qualquer navegador moderno para executar os exemplos e projetos.

Se preferir, você pode rodar um servidor local simples, por exemplo usando o VSCode com a extensão Live Server ou via Python:

```bash
# Com Python 3.x
python -m http.server
```

Depois abra no navegador `http://localhost:8000`.
